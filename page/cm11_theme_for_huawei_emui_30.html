CM11 theme for EMUI 3.0

About

This project is about making EMUI 3.0 look like Cyanogenmod 11 (CM11) within all the limitations that Huawei has restricted EMUI to. In other words, a CM11 theme for EMUI 3.0. The device I use is a Huawei Ascend P7, so all resources are xxhdpi/1080p

Source is available on Github. Please note that this theme is distributed under the Apache licence complying with the Cyanogenmod licence. Any work derived from this will have to be distributed under the same licence.

Features

  • Replaced stock status bar icons with CM11 ones
  • Holo Dark settings background
  • Dark status bar theme
  • CM11 quick settings icons
  • CM11/Holo style text fields, radio buttons, check boxes, sliders
  • Hidden NFC icon
  • CM11/AOSP nav bar button style
more to be added later

The basics

EMUI themes are distributed in the .hwt format. This format is nothing else than a ZIP file with a different extension. Within an extracted EMUI theme file are multiple folders and multiple files. You can omit any of these if you don't want to theme an element.

NameTypeDescription
com.android.contactsFileArchive of contacts app theming data
previewFolderContains preview images (as displayed in Themes app)
com.android.settingsFileArchive of settings app theming data
com.huawei.android.thememanagerFileArchive of Themes app theming data
wallpaperFolderContains homescreen and lockscreen wallpapers
com.android.systemuiFileArchive of theming data for status bar and system popups
description.xmlFileXML file containing information about the theme as displayed in the Themes app
theme.xmlFile
framework-resFileContains xml for colour status bar notification texts
Table 1. Summary of all files within a .hwt archive
