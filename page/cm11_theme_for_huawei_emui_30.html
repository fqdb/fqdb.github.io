CM11 theme for EMUI 3

About

This project is about making EMUI 3 look like Cyanogenmod 11 (CM11) within all the limitations that Huawei has restricted EMUI to. In other words, a CM11 theme for EMUI 3. The device I use is a Huawei Ascend P7, so all resources are xxhdpi/1080p

Source is available on Github. Please note that this theme is distributed under the Apache licence complying with the Cyanogenmod licence. Any work derived from this will have to be distributed under the same licence.

If you want to make your own EMUI 3 theme but don't know where to start: I'm writing a reference guide on it: How to make an EMUI 3.0 theme

Features

  • Replaced stock status bar icons with CM11 ones
  • Holo dark settings theme
  • Dark status bar theme
  • CM11 quick settings icons
  • CM11/Holo style text fields, radio buttons, check boxes, sliders
  • Hidden NFC and headphone icon
  • CM11/AOSP nav bar button style
  • Themed Dialer/Contacts/Messaging
more to be added later including all stock icons themed and a fully themed dialer.

Installation instructions

First install

  1. Place theme file in folder /SDCARD/HWThemes (internal storage/HWThemes)
  2. Open Themes app, go to tab 'Mine' and apply theme

Update

  1. Close themes app
  2. Go to folder /HWThemes and overwrite or delete old version
  3. Open Themes app and apply theme
  4. Reboot
  5. (To update preview images you have to clear the cache of the Themes app)
